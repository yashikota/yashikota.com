# マークダウンガイド

このブログでは、以下のマークダウン拡張機能が使用できます。

## コールアウト（GitHub風）

```markdown
> [!NOTE]
> これはノートです

> [!TIP]
> これはヒントです

> [!WARNING]
> これは警告です

> [!IMPORTANT]
> これは重要な情報です
```

表示例:

> [!NOTE]
> これはノートです

> [!TIP]
> これはヒントです

> [!WARNING]
> これは警告です

> [!IMPORTANT]
> これは重要な情報です

## リンクカード

通常のリンクは自動的にリンクカードとして表示されます。

```markdown
https://example.com
```

## コードブロック

````markdown
```js
// コードブロックはシンタックスハイライトされます
function hello() {
  console.log("Hello, world!");
}
```
````

表示例:

```js
// コードブロックはシンタックスハイライトされます
function hello() {
  console.log("Hello, world!");
}
```

## YouTube埋め込み

YouTubeのURLを貼り付けるだけで埋め込みプレーヤーが表示されます。

```markdown
https://www.youtube.com/watch?v=dQw4w9WgXcQ
```

## Twitter埋め込み

TwitterのURLを貼り付けるだけで埋め込みツイートが表示されます。

```markdown
https://twitter.com/user/status/123456789
```

## 見出し

```markdown
# 見出し1
## 見出し2
### 見出し3
#### 見出し4
##### 見出し5
###### 見出し6
```

## リスト

```markdown
- 箇条書き1
- 箇条書き2
  - ネストされた箇条書き1
  - ネストされた箇条書き2
- 箇条書き3

1. 番号付きリスト1
2. 番号付きリスト2
   1. ネストされた番号付きリスト1
   2. ネストされた番号付きリスト2
3. 番号付きリスト3
```

表示例:

- 箇条書き1
- 箇条書き2
  - ネストされた箇条書き1
  - ネストされた箇条書き2
- 箇条書き3

1. 番号付きリスト1
2. 番号付きリスト2
   1. ネストされた番号付きリスト1
   2. ネストされた番号付きリスト2
3. 番号付きリスト3

## テーブル

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 行1列1 | 行1列2 | 行1列3 |
| 行2列1 | 行2列2 | 行2列3 |
| 行3列1 | 行3列2 | 行3列3 |
```

表示例:

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 行1列1 | 行1列2 | 行1列3 |
| 行2列1 | 行2列2 | 行2列3 |
| 行3列1 | 行3列2 | 行3列3 |

## 引用

```markdown
> これは引用です。
> 複数行にまたがることもできます。
>
> 空行を挟むこともできます。
```

表示例:

> これは引用です。
> 複数行にまたがることもできます。
>
> 空行を挟むこともできます。


表示例:

## 画像


## 強調

```markdown
*斜体*
_斜体_
**太字**
__太字__
***太字かつ斜体***
___太字かつ斜体___
```

表示例:

*斜体*
_斜体_
**太字**
__太字__
***太字かつ斜体***
___太字かつ斜体___

## 取り消し線

```markdown
~~取り消し線~~
```

表示例:

~~取り消し線~~

## チェックボックス

```markdown
- [ ] 未チェックの項目
- [x] チェック済みの項目
```

表示例:

- [ ] 未チェックの項目
- [x] チェック済みの項目
